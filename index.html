Three Main Aims of Developing HTML5

  • Rich Multimedia Support: HTML5 aimed to provide better support for multimedia elements like video and audio without needing third-party plugins.
  • Enhanced Semantics: HTML5 introduced new semantic elements like header, footer, nav, article, section, and aside.
  • Cross-platform Compatibility: HTML5 aimed to provide a standardized platform for web development, reducing the need for browser-specific hacks and workarounds.

Example of Inline HTML

Span & Bold are example of inline elements. They are primarily used for styling on specific word or section of a block level element

They are named as 'Inline Elements' because they work on the same line as the block element they are within



Semantic HTML and its Importance

Semantic HTML refers to the pratice of using HTML elements to convey the meaning or purpose of the content enclose within the element

This is important due to the complexity of website. Lessens the need for comments and reduces the number of generic HTML elements, such as div, in your HTML

Examples Of 'figure' and 'image' Elements

Figure Element

The figure element is related to the main flow, but its position is independent of the main flow. It specifies self-contained content.

Image Element

The image element is used to display images from a decided source. Different styles can be applied, including the size of the image.

Examples Of 'main', 'section', and 'article' Elements

Main Element

The 'main' element represents the main content of a body element. All the content on this webpage is within the 'main' element.

Section Element

The 'section' tag is simply used to define any section(block) within a webpage. It can be used to define specific elements to that entire section as well.

Article Element

Generally used to define independent, self-contained content. Examples of this are blog posts, forum posts, etc.
